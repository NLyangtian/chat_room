
##项目名称：<br>
> 知心话吧<br>
##项目描述：<br>
> 支持注册、设密、登录、群聊、私聊、退出等功能<br>
##使用技术：<br>
+ Java基础<br>
+ Socket编程<br>
+ 多线程<br>
##项目功能：<br>
+ 注册<br>
+ 设置密码<br>
+ 登录<br>
+ 私聊<br>
+ 群聊<br>
+ 退出<br>
##项目实现：<br>
+ 创建客户端和服务端<br>
+ 客户端：分发任务，采用读写线程，将数据写入到服务器以及从服务器读取数据<br>
+ 服务端：采用线程池调度执行服务器与客户端业务处理<br>
##项目总结：<br>
+ 熟悉项目的开发流程（需求分析，设计，编码，测试，发布）<br>
+ 熟悉了Java的网络编程常用的API和步骤<br>
+ 对多线程有了更深入的了解<br>
+ 熟练Maven工具的使用<br>
##优化与扩展:<br>
+ 参数严格校验<br>
+ 异常信息处理<br>
